Project

Our project is designed to develop a real-time ECG (Electrocardiogram) monitoring system utilizing a Raspberry Pi as the central processing unit. This innovative system captures ECG readings from sensors connected via Bluetooth, ensuring a seamless and wireless data transfer process. The Raspberry Pi processes the raw ECG data, formatting it for accurate display and analysis.


To enhance user security, we have implemented a unique login ID for each device, guaranteeing that only authorized personnel can access sensitive health information. The processed ECG readings are then hosted on a dedicated website, which provides an intuitive and user-friendly platform for medical professionals and users to monitor heart activity remotely. The website is built using web development technologies such as HTML, CSS, and JavaScript, enabling clear and engaging visualization of the ECG data in real-time.


Additionally, our project aims to provide essential features like data logging and the ability to track historical ECG readings over time, which can be beneficial for medical assessments. Future enhancements could include developing a mobile application for convenient access on the go and implementing alert systems to notify users of any critical conditions detected in the ECG readings.


Overall, this ECG monitoring system not only promotes better health monitoring but also leverages modern technology to improve accessibility and security in medical data management.
